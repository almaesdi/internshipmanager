Add a electron dependecy to the current project:
    - vue add electron-builder

------------------------------------------------------------------------Tag 0.0.3-----------------------------------------------------------------------------------

Add bootstrap for project styling:
    - npm install --save @popperjs/core bootstrap

we have to add the following lines to main.js (vue) to share the library across every vue component

    import "bootstrap/dist/css/bootstrap.min.css"
    import "bootstrap"
